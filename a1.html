The Data Visualization project

Levelling up the data visualization !

For our project, we are majorly using Power Bi. In fact, we have first visualized our graphs using various python libraries like Pandas, Numpy, Matplotlib, and then implemented the same using power Bi. Power BI is a collection of software services, apps, and connectors that we have used to make unrelated sources of data into coherent, visually immersive, and interactive insights. Apart from that, we are also using a dataset from an online survey performed in 2021 by World Federation of mental Health. We have used different attributes from our dataset like Students age, their marital status, their CGPA, e.t.c. We are trying to gain insights through the connections between all of these attributes. Power Bi has made this very easy to implement.
